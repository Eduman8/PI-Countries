# Henry Countries

Henry Countries is a full-stack web application focused on exploring countries around the world, viewing geographical information, and creating tourist activities associated with one or multiple countries.

The project was originally developed as an individual bootcamp project and later refined with a stronger focus on UI polish, stability, deployment, and professional portfolio presentation while maintaining the original architecture and stack.

## Main Features

- Country search by name
- Filter countries by continent
- Alphabetical sorting
- Population sorting
- Country detail pages
- Tourist activity creation
- Association between activities and countries
- Responsive interface
- REST API integration
- PostgreSQL persistence
